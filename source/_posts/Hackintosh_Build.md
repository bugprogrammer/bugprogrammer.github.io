>鉴于很多小伙伴问我如何编译Hackintosh Bootloader以及kexts，今奉上博主的编译脚本。<!--more-->

# 如何使用

## 环境准备

* App Store安装Xcode11
* 输入以下命令并按提示安装命令行工具
```
xcode-select --install
```

## 脚本使用
```
bash -c "$(curl -fsSL https://raw.githubusercontent.com/bugprogrammer/hackintosh/master/Hackintosh_build.sh)"
```
